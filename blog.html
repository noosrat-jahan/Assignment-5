Important FAQ

What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a standard object model and programming interface for HTML documents that can be modified with javascript language. With DOM javascript can change all the HTML elements, HTML attributes, CSS styles in the page and also can remove all the existing HTML elements and attributes. Javascript can react to all existing HTML events in the page.

How do you select an element from the DOM?

There are multiple methods to select an element from the DOM. For example: getElementById, getElementsByClassName, getElementsByTagName, querySelector, querySelectorAll. Among these methods getElementById and querySelector methods are used to select single elements in HTML document and getElementsByClassName, getElementsByTagName and querySelectorAll methods can be used to select and manipulate multiple elements at the same time in HTML document.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is an event handling pattern that allows us to handle events at a higher level in the DOM tree. By event delegation the handling of an event is delegated to a different element instead of the actual element that received the event. Event delegation is a useful pattern because it allows us to write cleaner code and create fewer event listeners with similar logic.

How do you manipulate an element's attributes and styles using the DOM?

At first I have to select an html element by using any of the element selecting method such as getElementById or getElementsByClassName etc. Then in order to manipulate that element's attribute I have to use setAttribute, getAttribute or removeAttribute method by passing expected value. And in order to manipulate element's style I have to use classList.add or classList.remove method properly.
